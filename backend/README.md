# Job Hunter

A production-grade backend application that automates job discovery, AI-based resume matching, and application tracking.

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Docker
- n8n

## Current Features

- Health API
- PostgreSQL Integration
- Create Job API
- List Jobs API

## Roadmap

- AI Resume Matching
- Job Search Automation
- Google Sheets Export
- Dashboard
- Redis
- GitHub Actions
- Kubernetes
- Prometheus
- Grafana
